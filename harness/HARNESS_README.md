# X CoEmergence Harness (v1)

This harness runs experiment windows that probe for emergent structure
in AI behavior. It is the standalone version of the CoEmergence harness
originally prototyped under CoSteward.

Tracks:

- 01_ProbeEvo: prompts and scenarios.
- 02_HarnessEvo: metrics and rubrics.
- 03_ProtocolEvo: loop and window structure.
- 04_PopulationEvo: model and agent mix.
- 05_MetaTheoryEvo: theory and interpretation.

Windows live under:

  harness/windows/CoEmergence_Window_YYYYMMDDTHHMMSSZ/

Each window folder should contain:

- window_meta.md
- prompts_used.md
- rubric_used.md
- transcript.md
- notes.md
